# BTCTX Landing Page

Marketing site for BTCTX — deployed at https://btctx.us

## Stack

- Vite
- React
- TypeScript
- Tailwind v4

## Development

```bash
npm install
npm run dev
npm run build
```

## Deployment

The site auto-deploys to Cloudflare Pages on push to `main`.
